ANGULAR JS
--------------

angular.element($0) -----------and you now have access to all the good things:
angular.element($0).scope() ---------- will return the scope for the element.
angular.element(targetNode).isolateScope()---------- for isolated scope
angular.element($0).controller()----------- will give you the controller 
$($0).scope().$parent ---------- properties of the parent scope
$($0).scope().$parent.$parent------- properties of the parent parent scope
$($0).scope().$root ---------- root scope
angular.element($0).injector()------ returns the app’s injector, which you can then use to get access to other things
angular.element($0).scope().id like ng-model   ----- Gives value of element
